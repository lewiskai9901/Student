### 15. `application/place/PlaceBatchJobExecutor.java` ✨ 新建

**核心逻辑**:

```java
@Async("batchJobExecutor")
public void executeAsync(PlaceBatchJob job) {
    job.start();

    // 分批处理（每批100个）
    for (List<Long> batch : batches) {
        processBatch(job, batch);  // 每批独立事务 ✨
        job.updateProgress();
    }

    job.complete();
}

private void processBatch(PlaceBatchJob job, List<Long> placeIds) {
    for (Long placeId : placeIds) {
        transactionTemplate.execute(status -> {  // 独立事务 ✨
            try {
                // 处理单个场所
                job.markItemSuccess(placeId);
            } catch (Exception e) {
                job.markItemFailure(placeId, e.getMessage());
            }
        });
    }
}
```

**代码量**: +250行

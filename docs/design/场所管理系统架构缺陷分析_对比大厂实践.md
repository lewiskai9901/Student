## 🔴 严重缺陷（P0）- 对比大厂实践

### 缺陷1：继承模型设计错误 - **无法应对父节点变更**

#### 大厂实践分析

**AWS Organizations - Service Control Policies (SCPs)**
```yaml
# AWS的策略继承模型
Root Organization (OU-ROOT)
  ├─ Deny S3 DeleteBucket  # 根策略
  └─ Production OU
      ├─ Inherits: Deny S3 DeleteBucket  # 自动继承
      ├─ Additional: Allow EC2 All       # 附加策略
      └─ Account-A
          └─ Effective Policies:
              - Deny S3 DeleteBucket   # 继承自Root
              - Allow EC2 All          # 继承自Production OU
              - Deny RDS DeleteDB      # 账号级别策略
```

**核心设计**：
1. **策略不存储，实时计算** - 子节点不存储父策略的值，查询时动态合并
2. **父节点变更自动生效** - 修改Root策略，所有子节点立即生效
3. **明确的继承标识** - `InheritedFrom: OU-ROOT`

**Google Cloud - IAM Role Inheritance**
```typescript
// Google Cloud的权限继承
interface ResourceIAM {
  resource: string
  bindings: PolicyBinding[]
  inheritedBindings: PolicyBinding[]  // 明确区分继承的

  getEffectivePolicy(): Policy {
    // 实时计算有效策略
    return merge(
      this.parentResource?.getEffectivePolicy(),
      this.bindings
    )
  }
}
```

#### 正确设计（对标大厂）

```java
// ✅ 方案A：NULL表示继承（推荐，类似Kubernetes ConfigMap）
public class UniversalPlace {
    private Long orgUnitId;  // NULL = 继承，非NULL = 覆盖

    /**
     * 获取有效的组织ID（实时计算）
     */
    public Long getEffectiveOrgUnitId() {
        if (this.orgUnitId != null) {
            return this.orgUnitId;  // 明确指定
        }

        // 递归查找父级
        if (this.parent != null) {
            return this.parent.getEffectiveOrgUnitId();
        }

        return null;
    }

    /**
     * 是否继承组织
     */
    public boolean isOrgInherited() {
        return this.orgUnitId == null;
    }

    /**
     * 覆盖父级组织
     */
    public void overrideOrganization(Long orgUnitId) {
        if (orgUnitId == null) {
            throw new IllegalArgumentException("使用 clearOrgOverride() 来恢复继承");
        }
        this.orgUnitId = orgUnitId;
    }

    /**
     * 恢复继承
     */
    public void clearOrgOverride() {
        this.orgUnitId = null;
    }
}

// ✅ 方案B：独立继承标识字段（类似AWS Tag Inheritance）
public class UniversalPlace {
    private Long orgUnitId;
    private Boolean isOrgInheritEnabled;  // 是否启用继承

    public Long getEffectiveOrgUnitId() {
        if (!isOrgInheritEnabled) {
            return this.orgUnitId;  // 使用自身值
        }

        if (this.parent != null) {
            return this.parent.getEffectiveOrgUnitId();
        }

        return this.orgUnitId;  // 兜底
    }
}
```

```java
// ✅ 完整审计事件监听器
@Component
@RequiredArgsConstructor
public class PlaceAuditLogger {

    private final PlaceAuditLogMapper auditLogMapper;

    @TransactionalEventListener(phase = TransactionPhase.AFTER_COMMIT)
    public void onPlaceUpdated(PlaceUpdatedEvent event) {

        // 获取请求上下文
        RequestContext ctx = RequestContextHolder.getContext();

        PlaceAuditLog log = PlaceAuditLog.builder()
            // 唯一标识
            .eventId(UUID.randomUUID().toString())
            .requestId(ctx.getRequestId())

            // 资源信息
            .resourceType("PLACE")
            .resourceId(event.getPlace().getId())
            .resourceName(event.getPlace().getPlaceName())

            // 操作信息
            .eventName("UpdatePlace")
            .eventType("ApiCall")
            .eventSource("place-service")
            .eventTime(LocalDateTime.now())

            // 用户身份
            .userId(ctx.getUserId())
            .userName(ctx.getUserName())
            .userType(ctx.getUserType())
            .accessKeyId(ctx.getJwtTokenId())
            .sessionId(ctx.getSessionId())
            .mfaAuthenticated(ctx.isMfaAuthenticated())

            // 请求上下文
            .sourceIp(ctx.getSourceIp())
            .userAgent(ctx.getUserAgent())
            .referer(ctx.getReferer())
            .apiEndpoint(ctx.getRequestUri())
            .httpMethod(ctx.getHttpMethod())

            // 变更内容
            .requestParameters(ctx.getRequestBody())
            .responseElements(ctx.getResponseBody())
            .beforeSnapshot(toJson(event.getOldPlace()))
            .afterSnapshot(toJson(event.getNewPlace()))
            .changedFields(calculateChangedFields(event.getOldPlace(), event.getNewPlace()))

            // 元数据
            .isRollback(false)
            .reason(ctx.getChangeReason())
            .tags(ctx.getTags())

            // TTL（法律合规：审计日志保留7年）
            .ttlExpiresAt(LocalDateTime.now().plusYears(7))

            .build();

        auditLogMapper.insert(log);
    }

    /**
     * 计算变更字段
     */
    private List<FieldChange> calculateChangedFields(UniversalPlace oldPlace, UniversalPlace newPlace) {
        List<FieldChange> changes = new ArrayList<>();

        if (!Objects.equals(oldPlace.getOrgUnitId(), newPlace.getOrgUnitId())) {
            changes.add(new FieldChange(
                "orgUnitId",
                oldPlace.getOrgUnitId(),
                newPlace.getOrgUnitId()
            ));
        }

        if (!Objects.equals(oldPlace.getCapacity(), newPlace.getCapacity())) {
            changes.add(new FieldChange(
                "capacity",
                oldPlace.getCapacity(),
                newPlace.getCapacity()
            ));
        }

        // ... 其他字段

        return changes;
    }
}

/**
 * 请求上下文（从ThreadLocal获取）
 */
@Component
public class RequestContextHolder {

    private static final ThreadLocal<RequestContext> CONTEXT = new ThreadLocal<>();

    public static RequestContext getContext() {
        RequestContext ctx = CONTEXT.get();
        if (ctx == null) {
            ctx = new RequestContext();
            CONTEXT.set(ctx);
        }
        return ctx;
    }

    public static void clear() {
        CONTEXT.remove();
    }
}

/**
 * 请求拦截器 - 填充上下文
 */
@Component
public class AuditContextInterceptor implements HandlerInterceptor {

    @Override
    public boolean preHandle(HttpServletRequest request, HttpServletResponse response, Object handler) {
        RequestContext ctx = RequestContextHolder.getContext();

        // 生成requestId
        ctx.setRequestId(UUID.randomUUID().toString());

        // 用户信息
        Authentication auth = SecurityContextHolder.getContext().getAuthentication();
        if (auth instanceof CustomUserDetails) {
            CustomUserDetails user = (CustomUserDetails) auth.getPrincipal();
            ctx.setUserId(user.getUserId());
            ctx.setUserName(user.getUsername());
            ctx.setUserType("IAM_USER");
            ctx.setJwtTokenId(extractTokenId(request));
            ctx.setSessionId(request.getSession().getId());
            ctx.setMfaAuthenticated(user.isMfaAuthenticated());
        }

        // 请求信息
        ctx.setSourceIp(getClientIp(request));
        ctx.setUserAgent(request.getHeader("User-Agent"));
        ctx.setReferer(request.getHeader("Referer"));
        ctx.setRequestUri(request.getRequestURI());
        ctx.setHttpMethod(request.getMethod());

        return true;
    }

    @Override
    public void afterCompletion(HttpServletRequest request, HttpServletResponse response,
                                Object handler, Exception ex) {
        RequestContextHolder.clear();
    }

    private String getClientIp(HttpServletRequest request) {
        String ip = request.getHeader("X-Forwarded-For");
        if (ip == null || ip.isEmpty()) {
            ip = request.getHeader("X-Real-IP");
        }
        if (ip == null || ip.isEmpty()) {
            ip = request.getRemoteAddr();
        }
        return ip;
    }
}
```

**核心设计**：
1. **异步Job模式** - 提交Job后立即返回JobId，后台异步执行
2. **进度追踪** - 实时查询进度（成功数、失败数、进行中）
3. **失败详情** - 记录每个失败项的原因和错误码
4. **结果报告** - 生成CSV报告，包含所有成功/失败项
5. **幂等性** - 支持重试失败项，不会重复操作成功项
6. **优先级** - 支持Job优先级，高优先级先执行

```java
@Component
@RequiredArgsConstructor
public class PlaceBatchJobExecutor {

    private final PlaceRepository placeRepository;
    private final OrgUnitRepository orgUnitRepository;
    private final PlaceBatchJobRepository jobRepository;
    private final ThreadPoolTaskExecutor taskExecutor;

    /**
     * 异步执行Job
     */
    @Async("batchJobExecutor")
    public void executeAsync(PlaceBatchJob job) {
        try {
            // 1. 更新状态为RUNNING
            job.start();
            jobRepository.save(job);

            // 2. 获取待处理项（支持断点续传）
            List<Long> pendingIds = job.getPendingItemIds();

            // 3. 分批处理（每批100个）
            int batchSize = 100;
            for (int i = 0; i < pendingIds.size(); i += batchSize) {
                List<Long> batch = pendingIds.subList(
                    i,
                    Math.min(i + batchSize, pendingIds.size())
                );

                processBatch(job, batch);

                // 更新进度
                job.updateProgress();
                jobRepository.save(job);
            }

            // 4. 完成Job
            job.complete();
            jobRepository.save(job);

        } catch (Exception e) {
            job.fail(e.getMessage());
            jobRepository.save(job);
        }
    }

    /**
     * 处理单批（独立事务）
     */
    @Transactional(propagation = Propagation.REQUIRES_NEW)
    protected void processBatch(PlaceBatchJob job, List<Long> placeIds) {
        String operation = job.getOperation();
        Map<String, Object> params = job.getParameters();

        if ("BATCH_ASSIGN_ORG".equals(operation)) {
            Long orgUnitId = (Long) params.get("orgUnitId");

            // 批量查询（减少数据库调用）
            List<UniversalPlace> places = placeRepository.findAllById(placeIds);

            // 批量更新
            for (UniversalPlace place : places) {
                try {
                    place.setOrgUnitId(orgUnitId);
                    placeRepository.save(place);

                    job.markItemSuccess(place.getId());

                } catch (Exception e) {
                    job.markItemFailure(place.getId(), e.getMessage());
                }
            }
        }
    }
}

    private LocalDateTime createdAt;

        job.id = UUID.randomUUID().toString();
        job.operation = operation;
        job.status = JobStatus.PENDING;
        job.itemIds = itemIds;
        job.parameters = parameters;
        job.totalCount = itemIds.size();
        job.failureDetails = new HashMap<>();
        job.createdBy = getCurrentUserId();
        job.createdAt = LocalDateTime.now();
        return job;
    }

    /**
     * 开始执行
     */
    public void start() {
        if (this.status != JobStatus.PENDING) {
            throw new IllegalStateException("Job已经开始执行");
        }
        this.status = JobStatus.RUNNING;
        this.startTime = LocalDateTime.now();
    }

    /**
     * 标记项成功
     */
    public void markItemSuccess(Long itemId) {
        this.successCount++;
        this.runningCount--;
    }

    /**
     * 标记项失败
     */
    public void markItemFailure(Long itemId, String reason) {
        this.failureCount++;
        this.runningCount--;
        this.failureDetails.put(itemId, reason);
    }

    /**
     * 获取待处理项（支持断点续传）
     */
    public List<Long> getPendingItemIds() {
        Set<Long> processedIds = failureDetails.keySet();
        return itemIds.stream()
            .filter(id -> !processedIds.contains(id))
            .collect(Collectors.toList());
    }

    /**
     * 更新进度
     */
    public void updateProgress() {
        // 计算进度百分比
    }

    /**
     * 完成Job
     */
    public void complete() {
        this.status = JobStatus.COMPLETED;
        this.endTime = LocalDateTime.now();
    }

    /**
     * 失败Job
     */
    public void fail(String reason) {
        this.status = JobStatus.FAILED;
        this.endTime = LocalDateTime.now();
    }

    /**
     * 创建重试Job（只包含失败项）
     */
    public PlaceBatchJob createRetryJob() {
        List<Long> failedIds = new ArrayList<>(this.failureDetails.keySet());
        return create(this.operation, failedIds, this.parameters);
    }

    /**
     * 计算进度
     */
    public double getProgress() {
        if (totalCount == 0) return 0;
        return (double) (successCount + failureCount) / totalCount * 100;
    }
```

```java
@Component
@RequiredArgsConstructor
public class CapacityAlertScheduler {

    private final PlaceRepository placeRepository;
    private final RedisTemplate<String, Object> redisTemplate;

    /**
     * 每5分钟计算一次容量预警
     */
    @Scheduled(cron = "0 */5 * * * *")
    public void updateCapacityAlerts() {
        log.info("开始计算容量预警...");

        // 1. 数据库层面计算
        List<UniversalPlace> places = placeRepository.findHighOccupancyPlaces(0.8, 1000);

        // 2. 转换为预警对象
        List<CapacityAlert> alerts = places.stream()
            .map(this::toCapacityAlert)
            .sorted(Comparator.comparing(CapacityAlert::getOccupancyRate).reversed())
            .collect(Collectors.toList());

        // 3. 写入Redis（Set + ZSet双存储）
        String cacheKey = "capacity:alerts";
        String zsetKey = "capacity:alerts:sorted";

        // 存储完整列表
        redisTemplate.opsForValue().set(cacheKey, alerts, 10, TimeUnit.MINUTES);

        // 存储有序集合（支持范围查询）
        redisTemplate.delete(zsetKey);
        for (CapacityAlert alert : alerts) {
            redisTemplate.opsForZSet().add(
                zsetKey,
                alert,
                alert.getOccupancyRate()
            );
        }
        redisTemplate.expire(zsetKey, 10, TimeUnit.MINUTES);

        log.info("容量预警计算完成，共{}个预警", alerts.size());

        // 4. 发送通知（可选）
        sendAlertsToAdmins(alerts);
    }
```

```java
@Repository
public interface PlaceRepository extends JpaRepository<UniversalPlace, Long> {

    /**
     * 原子性增加占用数（只有容量足够时才成功）
     * @return 影响行数（1=成功，0=失败）
     */
    @Modifying
    @Query("UPDATE UniversalPlace p " +
           "SET p.currentOccupancy = p.currentOccupancy + 1 " +
           "WHERE p.id = :placeId " +
           "AND p.currentOccupancy < p.capacity")
    int incrementOccupancyIfAvailable(@Param("placeId") Long placeId);

    /**
     * 原子性减少占用数
     */
    @Modifying
    @Query("UPDATE UniversalPlace p " +
           "SET p.currentOccupancy = p.currentOccupancy - 1 " +
           "WHERE p.id = :placeId " +
           "AND p.currentOccupancy > 0")
    int decrementOccupancyIfPossible(@Param("placeId") Long placeId);
}

// 应用服务
@Service
@Transactional
public class PlaceOccupancyService {

    /**
     * 入住（原子操作，防止超卖）
     */
    public void checkIn(Long placeId, Long userId) {
        // 1. 先尝试原子性增加占用数
        int updated = placeRepository.incrementOccupancyIfAvailable(placeId);

        if (updated == 0) {
            // 容量已满或场所不存在
            UniversalPlace place = placeRepository.findById(placeId).orElseThrow();
            throw new IllegalStateException("容量已满: " + place.getCurrentOccupancy() + "/" + place.getCapacity());
        }

        // 2. 创建入住记录
        try {
            UniversalPlaceOccupant occupant = UniversalPlaceOccupant.create(placeId, userId);
            occupantRepository.save(occupant);

        } catch (Exception e) {
            // 回滚占用数
            placeRepository.decrementOccupancyIfPossible(placeId);
            throw e;
        }
    }

    /**
     * 退房（原子操作）
     */
    public void checkOut(Long occupantId) {
        // 1. 删除入住记录
        UniversalPlaceOccupant occupant = occupantRepository.findById(occupantId).orElseThrow();
        occupantRepository.deleteById(occupantId);

        // 2. 原子性减少占用数
        int updated = placeRepository.decrementOccupancyIfPossible(occupant.getPlaceId());

        if (updated == 0) {
            log.warn("退房时占用数已经为0，可能存在数据不一致");
        }
    }
}
```

```java
        int maxRetries = 3;
        for (int i = 0; i < maxRetries; i++) {
            try {
                return attemptCheckIn(placeId, userId);

            } catch (ObjectOptimisticLockingFailureException e) {
                if (i == maxRetries - 1) {
                    throw new IllegalStateException("并发冲突，请稍后重试");
                }
                // 短暂延迟后重试
                Thread.sleep(10 * (i + 1));
            }
        }

        placeRepository.save(place);

        // 5. 创建入住记录
```

**方案C：分布式锁（推荐，适合高冲突场景）**

```java
@Service
@RequiredArgsConstructor
public class PlaceOccupancyService {

    private final RedissonClient redissonClient;

    /**
     * 入住（使用分布式锁）
     */
    @Transactional
    public void checkIn(Long placeId, Long userId) {
        String lockKey = "place:lock:" + placeId;
        RLock lock = redissonClient.getLock(lockKey);

        try {
            // 1. 获取锁（最多等待10秒，锁自动释放时间30秒）
            boolean acquired = lock.tryLock(10, 30, TimeUnit.SECONDS);

            if (!acquired) {
                throw new IllegalStateException("系统繁忙，请稍后重试");
            }

            // 2. 查询场所
            UniversalPlace place = placeRepository.findById(placeId).orElseThrow();

            // 3. 检查容量
            if (place.getCurrentOccupancy() >= place.getCapacity()) {
                throw new IllegalStateException("容量已满");
            }

            // 4. 更新占用数
            place.setCurrentOccupancy(place.getCurrentOccupancy() + 1);
            placeRepository.save(place);

            // 5. 创建入住记录
            UniversalPlaceOccupant occupant = UniversalPlaceOccupant.create(placeId, userId);
            occupantRepository.save(occupant);

        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
            throw new IllegalStateException("操作被中断");

        } finally {
            // 6. 释放锁
            if (lock.isHeldByCurrentThread()) {
                lock.unlock();
            }
        }
    }
}
```

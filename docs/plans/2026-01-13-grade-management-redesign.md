# 年级管理界面重设计方案

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 重新设计年级管理界面，采用主从面板布局，支持拖拽配置专业，多彩主题区分年级

**Architecture:** Vue 3 Composition API + Tailwind CSS + HTML5 Drag & Drop API

**Tech Stack:** Vue 3, TypeScript, Tailwind CSS, Lucide Icons

---

## 设计决策

| 项目 | 决策 |
|------|------|
| 布局形式 | 主从面板（左侧年级列表 + 右侧配置详情） |
| 添加方式 | 拖拽式 + 点击添加双模式 |
| 展示风格 | 专业卡片墙，方向以标签形式内嵌 |
| 配色方案 | 多彩主题（不同年级不同主题色） |

## 多彩主题色

- 2024级：蓝色系 (#3B82F6 / blue-500)
- 2023级：紫色系 (#8B5CF6 / violet-500)
- 2022级：青色系 (#06B6D4 / cyan-500)
- 2021级：橙色系 (#F97316 / orange-500)
- 更早年级：灰色系 (#6B7280 / gray-500)

## 组件设计

### 1. GradeSelector（左侧年级选择器）

- 垂直堆叠的时间轴风格
- 每个年级卡片显示：年级名称、专业数量、方向数量、学生/班级数
- 选中状态：主题色高亮边框 + 浅色背景
- 未配置专业的年级显示警告图标

### 2. MajorConfigPanel（右侧配置面板）

- 顶部：年级信息 + 统计数据
- 中部：已配置专业的卡片网格
- 底部：可折叠的专业池

### 3. MajorCard（专业卡片）

- 卡片头部：专业名称 + 主题色渐变背景
- 卡片内容：方向标签列表（可移除）
- 卡片底部：统计信息 + 操作按钮

### 4. MajorPool（专业池）

- 可折叠设计，默认展开
- 搜索功能过滤专业
- 专业以小卡片形式展示，支持拖拽和点击添加

## 交互动效

1. **年级切换**：右侧内容淡出(150ms) → 淡入新内容(200ms)
2. **拖拽添加**：目标区域虚线边框闪烁，放置后弹跳效果
3. **删除专业**：消散动画 + 其他卡片平滑补位

## 空状态设计

- 毕业帽插画
- 提示文字："该年级尚未配置专业"
- 快捷操作：复制其他年级配置、添加专业

## 亮点功能

- **复制配置**：一键复制其他年级的全部专业配置
- **批量添加**：从专业池批量选择添加
- **拖拽排序**：已配置专业支持拖拽调整顺序

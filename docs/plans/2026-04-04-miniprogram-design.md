# 小程序端设计文档

## 技术栈

- **框架**: uni-app (Vue 3 + Vite + TypeScript)
- **UI 库**: wot-design-uni
- **状态管理**: Pinia
- **分页组件**: z-paging

## UI 风格: 钢蓝专业

### 设计规范

**色彩:**
- 主色: `#3a7bd5` (钢蓝)
- 警告色: `#e0592a`
- 成功色: `#15a87e`
- 辅助色: `#9a5cc6`, `#d4a030`
- 背景: `#f3f5f8`
- 卡片: `#ffffff`
- 主文字: `#1a2840`
- 次文字: `#5a6a7a`
- 辅助文字: `#a0aab4`

**字体:**
- 英文/数字: Plus Jakarta Sans
- 中文: Noto Sans SC (系统默认)
- 字重: 400(正文) / 500(标签) / 700(标题) / 800(大标题)

**圆角:**
- 卡片: 14px
- 按钮: 10px
- 图标容器: 14px
- 输入框: 10px
- TabBar 激活: 12px

**阴影:**
- 卡片: `0 2px 6px rgba(58,123,213,0.06)`
- 搜索: `0 1px 2px rgba(0,0,0,0.03)`
- TabBar: `0 -1px 3px rgba(0,0,0,0.03)`

**布局特征:**
- 4x2 白卡宫格模块入口
- 右侧工具按钮区(通知铃铛、扫码)
- Section 标题带 "全部 >" 链接
- 卡片式通知列表，左侧彩色方点状态指示
- 高信息密度，克制留白

## 认证流程

1. 微信授权登录 + 账号密码登录 (双通道)
2. 首次微信登录需绑定已有系统账号
3. Token 双令牌: accessToken (2h) + refreshToken (30d)
4. 请求拦截统一注入 Bearer Token, 401 自动刷新
5. 后端需新增: `POST /api/auth/wechat-login`, `POST /api/auth/bind-wechat`

## 权限驱动首页

- 登录时后端返回 `permissions[]` 权限码列表
- 前端 `modules.ts` 注册表: `{ key, label, icon, path }`
- 首页根据权限过滤注册表, 动态渲染功能宫格
- 新增模块只需: 建页面 + 加路由 + 注册表加一条
- 路由守卫校验页面访问权限

## 第一期范围 (基础骨架)

1. 项目初始化 (uni-app + Vue3 + TS + wot-design-uni + Pinia)
2. 请求封装 (request.ts, Token 注入/刷新/错误处理)
3. 登录页 (账号密码登录, 微信登录预留)
4. 首页 (权限驱动模块注册表 + 通知列表)
5. 消息页 (TabBar 占位)
6. 我的页 (头像/姓名/角色/退出登录)
7. 全局样式主题 (钢蓝专业配色规范)
8. 路由守卫 (未登录拦截 + 权限校验)

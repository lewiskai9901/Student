## 一、增强后的数据模型

### Major（专业）增强字段

```sql
ALTER TABLE majors ADD COLUMN IF NOT EXISTS major_category_code VARCHAR(20)
  COMMENT '专业大类代码(对应国家目录)' AFTER major_code;
ALTER TABLE majors ADD COLUMN IF NOT EXISTS enrollment_target VARCHAR(50) DEFAULT '初中毕业生'
  COMMENT '招生对象：初中毕业生/高中毕业生/社会人员' AFTER description;
ALTER TABLE majors ADD COLUMN IF NOT EXISTS education_form VARCHAR(30) DEFAULT '全日制'
  COMMENT '办学形式：全日制/非全日制/企业新型学徒制' AFTER enrollment_target;
ALTER TABLE majors ADD COLUMN IF NOT EXISTS lead_teacher_id BIGINT
  COMMENT '专业带头人(用户ID)' AFTER education_form;
ALTER TABLE majors ADD COLUMN IF NOT EXISTS lead_teacher_name VARCHAR(50)
  COMMENT '专业带头人姓名' AFTER lead_teacher_id;
ALTER TABLE majors ADD COLUMN IF NOT EXISTS approval_year INT
  COMMENT '专业审批年份' AFTER lead_teacher_name;
ALTER TABLE majors ADD COLUMN IF NOT EXISTS major_status VARCHAR(20) DEFAULT 'ENROLLING'
  COMMENT '专业状态：PREPARING筹建/ENROLLING招生中/SUSPENDED暂停招生/REVOKED已撤销' AFTER status;
```

### MajorDirection（专业方向）增强字段

```sql
ALTER TABLE major_directions ADD COLUMN IF NOT EXISTS enrollment_target VARCHAR(50)
  COMMENT '招生对象（覆盖专业默认值）' AFTER phase2_years;
ALTER TABLE major_directions ADD COLUMN IF NOT EXISTS education_form VARCHAR(30)
  COMMENT '办学形式（覆盖专业默认值）' AFTER enrollment_target;
ALTER TABLE major_directions ADD COLUMN IF NOT EXISTS certificate_names TEXT
  COMMENT '对应职业资格证书名称(JSON数组)' AFTER education_form;
ALTER TABLE major_directions ADD COLUMN IF NOT EXISTS training_standard VARCHAR(200)
  COMMENT '培养标准(如:一体化课程标准)' AFTER certificate_names;
ALTER TABLE major_directions ADD COLUMN IF NOT EXISTS cooperation_enterprise VARCHAR(200)
  COMMENT '校企合作单位' AFTER training_standard;
ALTER TABLE major_directions ADD COLUMN IF NOT EXISTS max_enrollment INT
  COMMENT '计划招生人数上限' AFTER cooperation_enterprise;
ALTER TABLE major_directions ADD COLUMN IF NOT EXISTS sort_order INT DEFAULT 0
  COMMENT '排序号' AFTER max_enrollment;
ALTER TABLE major_directions ADD COLUMN IF NOT EXISTS enabled TINYINT DEFAULT 1
  COMMENT '是否启用' AFTER sort_order;
```

## 二、增强后的领域模型

### Major 聚合根

```java
public class Major extends AggregateRoot<Long> {
    private String majorCode;            // 专业代码（国家目录编码）
    private String majorName;            // 专业名称
    private String majorCategoryCode;    // 专业大类代码 ★ 新增
    private Long orgUnitId;              // 承办系部
    private String description;          // 专业简介
    private String enrollmentTarget;     // 招生对象 ★ 新增
    private String educationForm;        // 办学形式 ★ 新增
    private Long leadTeacherId;          // 专业带头人 ★ 新增
    private String leadTeacherName;      // 专业带头人姓名 ★ 新增
    private Integer approvalYear;        // 审批年份 ★ 新增
    private MajorStatus majorStatus;     // 专业状态(筹建/招生/暂停/撤销) ★ 新增
    private Boolean enabled;
    private Integer sortOrder;
    private List<MajorDirection> directions;
}

// 专业状态枚举
public enum MajorStatus {
    PREPARING("筹建中"),   // 正在筹建，还未招生
    ENROLLING("招生中"),   // 正常招生
    SUSPENDED("暂停招生"), // 暂时停止招生
    REVOKED("已撤销");     // 专业被撤销
}
```

public class MajorDirection implements Entity<Long> {

    private Integer maxEnrollment;       // 计划招生人数上限

所有计分策略详解

点击策略卡片查看详细配置

基础策略 4个
✓
扣分制 常用
从基准分开始，发现问题扣除相应分数
得分 = 基准分 - Σ扣分
加分制 常用
从0分开始，完成项目累加分数
得分 = 0 + Σ加分
基准分制 常用
从基准分开始，可加可减
得分 = 基准分 ± Σ调整分
清单制 常用
按完成率计算，适合任务清单
得分 = 完成数 / 总数 × 100
等级策略 4个
等级制 常用
直接评定A/B/C/D/E等级
等级 → 预设分数映射
星级制 常用
1-5星评定，直观易用
得分 = 星数 × 20
合格制 常用
二元判断：合格/不合格
合格=100, 不合格=0
评语制 通用
优/良/中/差文字评定
优=90,良=75,中=60,差=40
高级策略 6个 · 折叠
阶梯计分 高级
不同严重程度对应不同扣分档位
轻微-1, 一般-3, 严重-5, 特别严重-10
累进计分 高级
累计次数影响分值，惩罚屡教不改
第n次扣分 = 基础分 × 2^(n-1)
倍率计分 高级
特殊时期/情况放大或缩小分值
实际扣分 = 基础扣分 × 倍率
封顶保底 高级
设置分数上限和下限
得分 = max(min(原始分, 上限), 下限)
一票否决 高级
触发关键项直接不合格
触发否决项 → 总分=0
门槛制 高级
必须达到门槛才参与评定
出勤率 < 80% → 不参评
时间策略 4个 · 折叠
时间衰减 高级
历史分数随时间降低权重
本周×1.0 + 上周×0.8 + 上上周×0.6
滚动计算 高级
取最近N期的平均分
得分 = (W1+W2+W3+W4) / 4
趋势加成 高级
考虑进步/退步趋势
环比上升+3分, 环比下降-2分
周期重置 高级
周期性从基准分重新开始
每月1号重置为100分
相对策略 4个 · 折叠
排名计分 高级
根据排名确定分数
第1=100, 第2=95, 第3=90...
强制分布 高级
按比例强制分配等级
A(10%) B(20%) C(40%) D(20%) E(10%)
标准分 高级
相对于平均值的标准分
Z = (原始分-均值)/标准差×10+50
百分位 高级
超过多少人的百分比
超过80%的人 = 80分
组合/特殊策略 5个 · 折叠
连续奖惩 高级
连续达标/不达标触发奖惩
连续5天满分+5, 连续3天不及格-10
多维组合 高级
多个维度独立计分再汇总
过程分×40% + 结果分×60%
条件触发 高级
满足条件才应用某规则
基础分 + (达标 ? 奖励 : 0)
抽查计分 高级
随机抽查，抽中才计入
抽中项计入，未抽中忽略
互评计分 高级
多人评分取平均或去极值
去掉最高最低取平均

数据模型设计

计分系统的完整数据结构

Template 检查模板
id Long 主键
templateName String 模板名称
templateCode String 模板编码
scoringStrategy Enum 主计分策略
baseScore Integer 基准分
minScore Integer 最低分（封底）
maxScore Integer 最高分（封顶）
defaultInputType Enum 默认打分方式
advancedRules JSON 高级规则配置
↓ 1:N
TemplateCategory 检查类别
id Long 主键
templateId Long 所属模板
categoryName String 类别名称
weight Integer 权重百分比
↓ 1:N
TemplateScoreItem 检查项
id Long 主键
categoryId Long 所属类别
itemName String 检查项名称
itemCode String 检查项编码
inputType Enum 打分方式
scoreOptions JSON 分值选项配置
isVeto Boolean 是否一票否决
maxDeduction Integer 单项封顶
progressiveRule JSON 累进规则
canLinkIndividual Boolean 可关联个人
requiresPhoto Boolean 需要拍照
requiresRemark Boolean 需要备注

JSON配置示例

// scoreOptions - 分值选项配置
{
  "options": [
    { "label": "合格", "value": 0 },
    { "label": "轻微问题", "value": -1 },
    { "label": "一般问题", "value": -3 },
    { "label": "严重问题", "value": -5 }
  ]
}

// progressiveRule - 累进规则配置
{
  "enabled": true,
  "baseValue": -1,
  "multiplier": 2,  // 每次翻倍
  "maxTimes": 5     // 最多累计5次
}

// advancedRules - 高级规则配置
{
  "capEnabled": true,
  "capMin": 0,
  "capMax": 100,
  "vetoEnabled": true,
  "progressiveEnabled": false,
  "consecutiveReward": {
    "enabled": true,
    "days": 5,
    "bonus": 5
  },
  "timeDecay": {
    "enabled": false,
    "weights": [1.0, 0.8, 0.6, 0.4]
  }
}
